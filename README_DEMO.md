# Silk Road Hub Demo / Offline

## Самый простой запуск (Windows, one-click)
1. Откройте корень проекта.
2. Дважды кликните `index.bat`.
3. Дождитесь сборки, браузер откроется автоматически.

Если браузер не открылся автоматически:
- откройте URL из вывода launcher (`http://127.0.0.1:<порт>/#/`)

## Запуск через npm
```bash
npm install
npm run demo
```

Доступные алиасы:
- `npm run demo:launch` (build + server + автооткрытие браузера)
- `npm run preview` (build + server)
- `npm run offline`

## Что делает preview
- выполняет production build (`next build`);
- запускает локальный `next start` сервер;
- сохраняет hash-навигацию `/#/...` и поднимает API endpoints (`/api/auth/*`, `/api/files/*`) для demo.

## Важно
- Не запускайте demo через `file://.../index.html`.
- Для корректной загрузки CSS/JS/ассетов нужен локальный сервер.

## Проверка, что все работает
- Открывается главная страница с нормальным стилем (не сырой HTML).
- Загружаются логотип и изображения (при отсутствии сети срабатывает fallback).
- Переходы `/#/...` работают.
- Формы/кнопки кликабельны.
